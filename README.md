# Coding Exercises Repository

Welcome to this collection of coding exercises! This repository is a personal playground for exploring various data structures, algorithms, and problem-solving techniques.

Here you'll find:

*   **Complete Data Structures:** Implementations of common data structures and algorithms.
*   **Cracking the Coding Interview:** Solutions to problems from the famous "Cracking the Coding Interview" book.
*   **LeetCode:** Solutions to LeetCode problems, categorized by topic.

Feel free to browse, learn, or use it as a reference!
